Architecture

12-step pipeline: Input Guardrails → Cache Check → Query Rewrite → Multi-Query Expansion → Embedding → Chunk Cache Check → Parallel Vector Search → Cross-Encoder Reranking → Agentic Retrieval Loop → Output Guardrails → Evaluation → Response

Key features:

  • Structure-aware chunking (markdown headings, paragraphs, fixed-size fallback)
  • Two-level caching (user query cache + chunk embedding cache)
  • Cross-encoder reranking for precision
  • Agentic retrieval with self-reflection and hallucination checking
  • Full tracing/observability on every pipeline step
  • Input/output guardrails with PII detection and redaction
  • LLM-as-judge evaluation (faithfulness, relevance, completeness)
